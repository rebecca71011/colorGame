Guide:

  • The RGB color model is an additive color model in which red, green and blue light are added together in various ways to reproduce a broad array of colors.

  • The RGB color values you are tasked to identify are not simple colors like rgb(255 , 0, 0) (i.e. red) or rgb(0, 0, 0) (i.e. black), it will be colors such as rgb(209, 61, 209), which is a pinkish- purplish colour.
